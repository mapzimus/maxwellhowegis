// About

I'm Maxwell Howe — a GIS professional and builder with an MS in Geo-Information Science from Salem State University (2025). I work across spatial analysis, R and Python scripting, and web development to build tools that make geographic data useful — from spatial analysis pipelines to full-stack web applications.

I teach high school math and statistics, which shapes how I approach GIS. Most of this work is translation — turning complex spatial data into something a non-technical audience can actually act on. More about me →
