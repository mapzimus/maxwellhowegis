How it works

Draw features on the map (or import a GeoJSON file), then hit Bake PMTiles. The browser slices your geodata into a pyramid of tiles, encodes them as Mapbox Vector Tile protobuf, and assembles a PMTiles v3 archive — entirely locally, with no server. Download the archive, or publish it to cloud storage and share a streaming URL that anyone can open on any device. The viewer streams only the tiles it needs via HTTP range requests — no full download required.
