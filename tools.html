Web Tools & Apps

Interactive tools and applications I've built. New tools are added as they're developed — check back often.

TrueScale

Drag any country, state, or province across the map to compare its real size. The Mercator projection stretches land near the poles — Greenland looks as big as Africa, but it's 14× smaller. TrueScale shows you by how much. No ads, no tracking.

Live

TappyMaps

Lightweight US-state map maker — tap states to color them, add legends, themes, north arrows, and scale bars. County-level detail and CSV-driven data maps in Pro.

Live

Geopuesto

Antipodal point calculator — enter any location and instantly find its exact geographic opposite on the other side of Earth.

Live

Viral Map Idea Generation

Custom scoring engine that ranks ~20,000 candidate map ideas by predicted virality. Weighted V-Score across seven dimensions, data-readiness penalty, and format bonuses.

Live

PocketTiles Studio

Draw features (or import GeoJSON), bake them into a cloud-native PMTiles archive entirely in the browser, and share a streaming URL — no tile server, no local storage bloat.
