@Mapzimus Idea Board

A custom platform for managing, scoring, and correlating viral infographic map ideas. Built from scratch to systematize the creative process behind maps that go viral.

Launch Board View on GitHub
// What It Does

A Systematic Approach to Viral Maps

Most viral maps happen by accident. Mapzimus makes it repeatable by scoring ideas across multiple dimensions before a single pixel is placed.

01 Multi-Dimensional Scoring

Every idea is scored on V-Score, emotional impact, clarity, surprise, and more. Drag sliders to filter by score ranges and stack multiple dimensions for precision targeting.

02 Geographic Filtering

Filter ideas by state-level geography to find story angles tied to specific regions. Useful for targeting local audiences or finding patterns unique to certain areas.

03 Data Correlation Engine

Pick any variable and find every idea it connects to — both direct usage and joinable relationships. Reveals hidden connections between datasets and story angles.

04 Format Analysis

Browse ideas by output format — scatter plots, choropleth maps, bivariate visualizations, bar charts, treemaps — to match the right visual form to each story.

05 Type Classification

Ideas are classified as MAP, XREF, CHART, or RANK — each with distinct visual treatment. Filter by type to focus on specific visualization approaches.

06 Status Tracking

Track each idea from initial concept through In Progress, Built, and Published stages. Notes and annotations persist for ongoing editorial context.

// Tech Stack

Built From Scratch

No frameworks, no dependencies. Pure HTML, CSS, and JavaScript with the Web Crypto API for access control.

Vanilla JS

Zero framework overhead. Custom state management, dynamic rendering, and real-time filtering without React, Vue, or Angular.

CSS Custom Properties

Full dark theme with semantic color tokens, type-specific badge colors, and responsive layouts using CSS Grid and Flexbox.

Web Crypto API

SHA-256 password hashing for access control. Session persistence via sessionStorage. No backend required.

GitHub Pages

Hosted for free on GitHub Pages with instant deploys on push. No servers to maintain, no hosting costs.
